# Git credentials

GitHub credentials are available in environment variables when running in the Aria Chat desktop app:
- `GIT_USERNAME` — GitHub username configured in Settings > General
- `GIT_PASSWORD` — GitHub personal access token configured in Settings > General

ALWAYS use these credentials for git push. The correct command is:
  git push "https://$env:GIT_USERNAME:$env:GIT_PASSWORD@$(git remote get-url origin)" HEAD:<remote-branch>
Never run `git push` without the credential-embedded URL — it will fail.
